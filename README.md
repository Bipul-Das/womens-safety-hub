## 💡 Project Overview
**Women's Safety Hub** is a single-page application (SPA) architecture built to provide immediate assistance in distress situations while fostering long-term empowerment through legal education. The platform focuses on speed, accessibility, and data privacy.

### 🌟 Key Features

#### 🚨 1. Emergency Response Engine
* **One-Click SOS:** instantly acquires GPS coordinates and simulates broadcasting alerts to trusted contacts.
* **Rapid Dial Directory:** High-contrast, direct-dial interface for Police (100), Women's Helpline (1091), and Cyber Crime Units.
* **Contact Management:** CRUD functionality to manage trusted contacts (Parents, Friends) using LocalStorage.

#### ⚖️ Legal Literacy Framework
* **Rights Database:** Simplified, plain-language breakdown of critical Indian laws (Zero FIR, POSH Act, Domestic Violence Act).
* **Real-Time Search:** Instant filtering engine to help users find relevant legal protections in seconds.
* **Accordion Interface:** Clean UI to present dense legal text in a digestible format.

#### 🛡️ Advanced Security & Privacy
* **Quick Exit Trigger:** A "Panic Button" that instantly redirects the browser to a neutral site (Google) to protect users in domestic distress.
* **Session Security:** Industry-standard dashboard to monitor active sessions and device history.
* **Privacy-First:** Utilizes `LocalStorage` to keep sensitive contact data on the user's device, minimizing cloud risks.
